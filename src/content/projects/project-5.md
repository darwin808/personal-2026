A blazingly fast application launcher for macOS, built in Rust as a modern alternative to Spotlight and Rofi. Provides instant search and launch capabilities with minimal resource usage.

Designed for developers who want a lightweight, customizable launcher with the performance benefits of native Rust code. Features keyboard-driven navigation and extensible plugin architecture.
